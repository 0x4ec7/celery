Celery - Distributed Task Queue¶

Celery aims to be a fast, reliable and flexible, best-of-breed solution to process vast amounts of messages in a distributed fashion, while providing operations with the tools to maintain such a system.

Celery is Open Source and licensed under the BSD License.

Getting Started¶

  • If you are new to Celery you can get started by following the First steps with Celery tutorial.
  • You can also check out the FAQ.
