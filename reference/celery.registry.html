Task Registry - celery.registry¶

celery.registry

class celery.registry.TaskRegistry¶

Site registry for tasks.

AlreadyRegistered()¶
The task is already registered.
NotRegistered()¶
The task is not registered.
autodiscover()¶
Autodiscovers tasks using celery.discovery.autodiscover().
filter_types(type)¶
Return all tasks of a specific type.
get_all()¶
Get all task types.
get_all_periodic()¶
Get all periodic task types.
get_all_regular()¶
Get all regular task types.
get_task(name)¶
Get task by name.
register(task, name=None)¶

Register a task in the task registry.

Task can either be a regular function, or a class inheriting from celery.task.Task.

Parameters:
  • name – By default the Task.name attribute on the task is used as the name of the task, but you can override it using this option.
Raises AlreadyRegistered:
 

if the task is already registered.
unregister(name)¶

Unregister task by name.

Parameters:
  • name – name of the task to unregister, or a celery.task.Task class with a valid name attribute.
Raises celery.exceptions.NotRegistered:
 

if the task has not been registered.
