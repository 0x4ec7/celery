Periodic Task Schedule Behaviors - celery.task.schedules¶

class celery.task.schedules.crontab(minute=None, hour=None, day_of_week=None, nowfun=<built-in method now of type object at 0x102ade5c0>)¶

A crontab can be used as the run_every value of a PeriodicTask to add cron-like scheduling.

Like a cron job, you can specify units of time of when you would like the task to execute. While not a full implementation of cron’s features, it should provide a fair degree of common scheduling needs.

You can specify a minute, an hour, and/or a day of the week.

minute¶
An integer from 0-59 that represents the minute of an hour of when execution should occur.
hour¶
An integer from 0-23 that represents the hour of a day of when execution should occur.
day_of_week¶
An integer from 0-6, where Sunday = 0 and Saturday = 6, that represents the day of week that execution should occur.
