Clock Service - celery.beat¶

class celery.beat.ScheduleEntry(name, last_run_at=None, total_run_count=None)¶

An entry in the scheduler.

Parameters:
  • task – The task class.
  • last_run_at – The time and date when this task was last run.
  • total_run_count – Total number of times this periodic task has been executed.
class celery.beat.Scheduler(**kwargs)¶

Scheduler for periodic tasks.

Parameters:
  • registry – The task registry to use.
  • schedule – The schedule dictionary. Default is the global persistent schedule celery.beat.schedule.
get_due_tasks()¶
Get all the schedule entries that are due to execution.
schedule_registry()¶
Add the current contents of the registry to the schedule.
tick()¶
Run a tick, that is one iteration of the scheduler. Executes all due tasks.
exception celery.beat.SchedulingError¶
