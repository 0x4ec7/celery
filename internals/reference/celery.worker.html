celery.worker¶

celery.worker.WORKER_SIGIGNORE¶

List of signals to ignore when a child process starts.

celery.worker.WORKER_SIGRESET¶

List of signals to reset when a child process starts.

class celery.worker.WorkController(concurrency=None, logfile=None, loglevel=None, send_events=None, hostname=None, ready_callback=<function noop at 0x1049000c8>, embed_clockservice=False, pool_cls=None, consumer_cls=None, mediator_cls=None, eta_scheduler_cls=None, schedule_filename=None, task_time_limit=None, task_soft_time_limit=None, max_tasks_per_child=None, pool_putlocks=None, db=None, prefetch_multiplier=None, eta_scheduler_precision=None, queues=None, disable_rate_limits=None, autoscale=None, autoscaler_cls=None, scheduler_cls=None, app=None)¶

Unmanaged worker instance.

concurrency¶

The number of simultaneous processes doing work (default: CELERYD_CONCURRENCY)

consumer¶

Consumer instance.

embed_clockservice¶

If True, celerybeat is embedded, running in the main worker process as a thread.

logfile¶

The logfile used, if no logfile is specified it uses stderr (default: CELERYD_LOG_FILE).

logger¶

The logging.Logger instance used for logging.

loglevel¶

The loglevel used (default: logging.INFO)

mediator¶

Instance of celery.worker.controllers.Mediator.

on_timer_error(exc_info)¶
on_timer_tick(delay)¶
pool¶

The pool instance used.

process_task(request)¶

Process task by sending it to the pool of workers.

ready_queue¶

The internal queue object that holds tasks ready for immediate processing.

send_events¶

Enable the sending of monitoring events, these events can be captured by monitors (celerymon).

start()¶

Starts the workers main loop.

stop(in_sighandler=False)¶

Graceful shutdown of the worker server.

terminate(in_sighandler=False)¶

Not so graceful shutdown of the worker server.

celery.worker.process_initializer(app, hostname)¶

Initializes the process so it can be used to process tasks.

Used for multiprocessing environments.
